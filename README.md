## About

26 years old. I break systems to understand them — then build better defenses.

For my bachelor thesis I conducted a full manual penetration test of a real startup's web application using OWASP WSTG and NIST SP 800-115. No automated scanners. The goal was finding what tools miss. The report included real findings, CVSS scores and remediation steps that the company acted on.

I also built a GUI for Volatility 3 — a memory forensics tool used in professional incident response — in collaboration with a cybersecurity company. Powerful tools should not require a manual.

Now I build the things I describe in interviews. Every project here has real code, real output and a reason it exists.

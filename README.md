## About

I work on the offensive and defensive sides of security. My focus is understanding how attacks actually work — then using that to build better detection, stronger controls and more realistic risk assessments.

Currently going deep on detection engineering, cloud attack paths and cryptographic weaknesses in real-world systems.

## Projects

**[soc-project](https://github.com/Speed-boo3/soc-project)**
Log analysis and alert detection pipeline built in Python. Detection rules are mapped to MITRE ATT&CK techniques and source IPs are checked against AbuseIPDB in real time. Includes an incident response playbook and a terminal dashboard.

**[grc-project](https://github.com/Speed-boo3/grc-project)**
Risk scoring engine with a network scanner that converts exposed ports into risk register entries. Compliance mapped to ISO 27001 and NIST CSF. Generates weekly reports with charts.
